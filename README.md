# toggle-password
Password visibility toggle UI built using HTML, CSS, and JavaScript. Clicking the eye icon shows or hides the password input.
# Password Toggle | Era21Tech

## Features
- Show/Hide password on click
- Clean UI with centered design
- Lightweight and beginner-friendly
- Uses simple JavaScript toggle logic

## Files
- `index.html` — Input field and toggle script.
- `style.css` — Styling for layout and input UI.

## How to Use
1. Download the project files.
2. Open `index.html` in any browser.
3. Type a password and click the eye icon to show/hide it.

## Customization
- Replace the eye icons in `/images/`
- Change colors and sizes from `style.css`
- Modify placeholder text in `index.html`
